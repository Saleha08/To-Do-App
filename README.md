# ✅ To-Do App

A simple and interactive To-Do List app built using HTML, CSS, and JavaScript. 📝✨

## 🚀 Features
- ➕ Add new tasks
- 🗑️ Delete tasks

## 🛠️ Technologies Used
- 🌐 **HTML** - Structure
- 🎨 **CSS** - Styling
- ⚡ **JavaScript** - Functionality
